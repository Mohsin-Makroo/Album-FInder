# 🎵 Album Finder (Spotify API)

A simple React app built with **Vite** that lets users search for music albums using the **Spotify Web API**.
 It displays album details like cover art, artist and a direct link to Spotify.

---

## 🚀 Features
- 🔎 Search albums by **artist name** or **album title**
- 🎨 Displays **album cover, artist,**
- 🌐 Open albums directly in **Spotify**
- ⚡ Built with **React + Vite**
- 🔑 Uses **Spotify API (Client Credentials Flow)**

---

## 🛠️ Tech Stack
- **React + Vite**
- **Spotify Web API**
- **CSS / Tailwind**
